# Netflix Clone (Frontend UI)

This project is a frontend UI clone inspired by Netflix,
built using only HTML and CSS. The focus of this project
is on layout, responsiveness, and visual design.

## Features
- Netflix-style homepage UI
- Responsive layout using CSS
- Clean and modern design

## Tech Stack
- HTML
- CSS

## Project Type
Frontend UI Development Project
